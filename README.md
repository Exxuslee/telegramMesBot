**Телеграм-бот уведомлений токов штанг коксовыталкивателей АКХЗ**

Данные берутся из локального OPC-сервера

## Starting
To view, just run:
```
node index
```

## Features
1. Connect to OPC-server
2. Создание осциллограммы тока
3. Интеграция в MES
4. Запуск, как службы либо, как приложения
